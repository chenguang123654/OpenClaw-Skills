# 弹道计算技能 (Ballistics Calculator)

专业弹道计算工具，支持多种射击场景的参数计算。

## 🏹 应用场景

- **弹弓发射** - 计算最佳射击角度和落点
- **弓箭抛射** - 远程射击角度修正
- **气枪计算** - 弹道下坠补偿
- **玩具枪弹道** - 抛物线轨迹预测

## 📐 支持的计算类型

| 类型 | 功能 | 示例 |
|------|------|------|
| **基本弹道** | 已知初速+角度→计算落点 | "初速50m/s，角度45度" |
| **仰角求解** | 已知初速+距离→求射击角度 | "初速60m/s，打20米要多少角度" |
| **高程修正** | 有高度差→弹道修正 | "从5米高打10米外的目标" |
| **能量计算** | 皮筋参数→弹丸能量 | "8mm钢珠，18股皮筋" |
| **完整轨迹** | 生成完整弹道数据表 | "输出完整弹道表" |

## 🧮 物理公式

```
重力加速度: g = 9.81 m/s²

水平距离:   x = v₀ × cos(θ) × t
垂直高度:   y = v₀ × sin(θ) × t - ½gt²
飞行时间:   t = 2 × v₀ × sin(θ) / g
最大高度:   h = v₀² × sin²(θ) / (2g)

抛物线近似射程: R ≈ v₀² × sin(2θ) / g
```

## 📖 使用方式

直接告诉我参数，支持多种表达方式：

```
"初速50米/秒，角度45度，计算落点"
"速度80，距离20米，求射击角度"
"从10米高度以30度发射，初速60米/秒"
"8mm钢珠，18股皮筋，能量多少"
```

## 🔧 高级功能

### 能量计算（弹弓）
```
钢珠直径 → 计算质量和体积
皮筋股数 → 估算弹性势能
输出: 焦耳、m/s、g.mm/s
```

### 轨迹预测
```
输出每0.1秒的位置点
格式: 时间(s) | 水平距离(m) | 垂直高度(m)
可生成: JSON / 表格 / 图表数据
```
